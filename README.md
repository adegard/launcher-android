# Minimalist Launcher

A simplified Android launcher built with Kotlin and Jetpack Compose.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Build System**: Gradle

## Requirements

- Android Studio Hedgehog or later
- Android SDK 35 (compile) / SDK 24+ (minimum)
- Kotlin 1.9+
- Java 11+
- Gradle 8.0+
